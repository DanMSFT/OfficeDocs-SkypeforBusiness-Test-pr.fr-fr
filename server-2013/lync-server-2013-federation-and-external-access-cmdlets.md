# Cmdlets de fédération et d’accès externe dans Lync Server 2013

 

_**Dernière rubrique modifiée :** 2016-12-08_

La fédération et l’accès externe fournissent deux fonctionnalités importantes : la fédération permet aux utilisateurs de communiquer avec des personnes se trouvant en dehors de votre organisation, alors que l’accès externe permet aux utilisateurs de se connecter à Microsoft Lync Server 2013 sans être dans le réseau interne. Les applets de commande disponibles pour la gestion de la fédération et de l’accès externe dans Lync Server 2013 vous permettent de déterminer avec qui vos utilisateurs peuvent ou ne peuvent pas communiquer, mais aussi de définir si ces utilisateurs peuvent se connecter à Lync Server sans devoir ouvrir une session sur le réseau interne.

## Applets de commande de fédération et d’accès externe

La plupart des tâches de gestion qui s’appliquent à la fédération et à l’accès externe peuvent être réalisées depuis le Panneau de configuration Lync Server. Ces mêmes tâches peuvent être effectuées via des applets de commande à partir de Lync Server Management Shell ou depuis un script ; l’utilisation d’un script vous permet d’automatiser certaines tâches.
