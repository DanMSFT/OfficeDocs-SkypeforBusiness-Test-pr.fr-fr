# Cmdlets de connectivité PSTN

 

_**Dernière rubrique modifiée :** 2016-12-08_

Voix Entreprise fournit des paramètres permettant d’assurer l’acheminement des appels depuis et vers le réseau téléphonique commuté (PSTN) sans jamais altérer la qualité du service. Vous pouvez configurer ces paramètres à l’aide des applets de commande disponibles dans Lync Server Management Shell.
