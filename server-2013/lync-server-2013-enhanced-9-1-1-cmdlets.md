# Cmdlets 9-1-1 améliorées

 

_**Dernière rubrique modifiée :** 2016-12-08_

Microsoft Lync Server 2013 intègre des cmdlets vous permettant de gérer la mise en œuvre Enhanced 9-1-1 (E9-1-1) d’une solution Voix Entreprise. Utilisez ces cmdlets pour mapper des points de connexion à des adresses physiques et configurer les paramètres nécessaires afin que les utilisateurs Voix Entreprise puissent passer des appels d’urgence et envoyer automatiquement leur position aux services d’urgence. Vous ne pouvez pas configurer E9-1-1 à partir du Panneau de configuration Lync Server ; vous devez utiliser des cmdlets.
