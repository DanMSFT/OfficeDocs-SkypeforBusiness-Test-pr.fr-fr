# Cmdlets des conférences Web

 

_**Dernière rubrique modifiée :** 2016-12-08_

Les conférences et les réunions en ligne sont des éléments importants dans Microsoft Lync Server 2013. Les cmdlets CsConferencingConfiguration et CsConferencingPolicy sont les principaux outils d’administration utilisés pour gérer les conférences à l’aide de Windows PowerShell.

## Cmdlets des conférences Web

Les cmdlets [New-CsClientPolicy](new-csclientpolicy.md) et [Set-CsClientPolicy](set-csclientpolicy.md) permettent également de configurer et de gérer Lync Server 2013.
